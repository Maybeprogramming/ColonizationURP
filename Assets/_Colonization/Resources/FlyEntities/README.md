# Fly Entity Shader

Шейдер для спрайтовых летающих сущностей (бабочек) с процедурной анимацией крыльев. Выполняет взмах крыльев на GPU без использования анимационных костей или blend shapes.

## Использование в проекте

### 1. Подготовка

1. Создайте или выберите **Material** в папке `Resources/FlyEntities/`.
2. В инспекторе материала выберите шейдер `Custom/FlyEntity`.
3. Присвойте текстуру спрайта в слот `_MainTex`.

### 2. Настройка через компоненты

Материал используется компонентом `Butterfly`. При инициализации бабочки (`Initialize()`) шейдер получает через `MaterialPropertyBlock`:

| Property | Описание |
|---|---|
| `_MainTex` | Текстура спрайта (задаётся через PropertyBlock) |
| `_Alpha` | Глобальная прозрачность (0 — невидима, 1 — полностью видна) |
| `_Scale` | Масштаб модели (умножает `vertex.xyz` перед трансформацией) |

## Параметры материала (Inspector)

### Main Tex
`2D` — **Текстура спрайта** бабочки. Ожидается спрайт-лист с разбивкой на 3 сегмента по UV.x (левое крыло, тело, правое крыло).

### Flap Speed
`Range(1, 20)` — **Скорость взмаха крыльев**. Частота синусоидальной анимации.

### Flap Angle
`Range(0, 1)` — **Угол взмаха крыльев**. Амплитуда поворота крыла.

### Global Alpha
`Range(0, 1)` — **Глобальная прозрачность**. Управляется из кода для fade-in/fade-out.

## Алгоритм работы

### Vertex Shader (`vert`)
Для каждой вершины спрайта:

1. **Масштабирование**: локальная позиция умножается на `_Scale`.
2. **Определение зоны крыла** по `uv.x`:
   - `uv.x < 1/3` → левое крыло: взмах в положительную сторону.
   - `uv.x > 2/3` → правое крыло: взмах в отрицательную сторону.
   - Иначе (центр, тело бабочки) — без анимации.
3. **Угол взмаха**: `sin(_Time.y * _FlapSpeed) * _FlapAngle`.
4. **Поворот в 2D (XZ)**: крыло поворачивается вокруг оси Z через матрицу 2D-поворота:
   ```
   x' = x * cos(angle) - y * sin(angle)
   y' = x * sin(angle) + y * cos(angle)
   z' = z
   ```
5. **Трансформация в экранное пространство**: `TransformObjectToHClip`.

### Fragment Shader (`frag`)
- Выборка текстуры: `SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, uv)`.
- Умножение альфа-канала на `_Alpha`.
- Отсечение (`clip`) пикселей с альфой < 0.01.

### Особенности
- **Cull Off** — отрендериваются обе стороны каждого треугольника.
- **ZWrite Off**, **Blend SrcAlpha OneMinusSrcAlpha** — стандартный прозрачный блендинг.
- **Render Queue = Transparent** — рендерится после непрозрачных объектов.

## Использованные технологии

- **Vertex Shader анимация** — взмах крыльев вычисляется на CPU-независимом этапе, без использования анимации Unity.
- **Матрица 2D-поворота** — поворот вокруг оси в плоскости экрана.
- **UV-сегментация спрайта** — деление текстуры на 3 зоны по `uv.x` для разных частей модели.
- **URP Transparent pipeline** — прозрачный рендеринг с `Blend` и `ZWrite Off`.
- **MaterialPropertyBlock** — уникальные параметры на каждый экземпляр (текстура, альфа, масштаб) без создания копий материала.
